The TypeScript SDK provides the same clean experience as the Python SDK - just install and use, no servers required.

<Info>
  The TypeScript SDK uses LangGraph.js under the hood, providing the same architecture as the Python version.
</Info>

## Installation

<CodeGroup>
```bash npm
npm install @delve/taxonomy
```

```bash yarn
yarn add @delve/taxonomy
```

```bash pnpm
pnpm add @delve/taxonomy
```
</CodeGroup>

## Quick Start

```typescript
import { Delve } from '@delve/taxonomy';

// Create client
const delve = new Delve({
  maxNumClusters: 5,
  useCase: 'Categorize customer feedback',
});

// Generate taxonomy
const result = await delve.run([
  'The app crashes when I click submit',
  'Please add dark mode support',
  'How do I reset my password?',
]);

// Access results
console.log('Categories:', result.taxonomy);
console.log('Labeled docs:', result.labeledDocuments);
```

## Configuration

```typescript
const delve = new Delve({
  // Model configuration
  model: 'claude-sonnet-4-5-20250929',        // Main model for reasoning
  fastLlm: 'claude-haiku-4-5-20251001',       // Fast model for summarization

  // Processing options
  sampleSize: 100,                             // Documents to sample for LLM labeling
  batchSize: 200,                              // Batch size for clustering
  maxNumClusters: 5,                           // Maximum categories to generate

  // Use case
  useCase: 'Categorize customer feedback',     // Guides taxonomy generation

  // API keys (optional - uses env vars by default)
  anthropicApiKey: process.env.ANTHROPIC_API_KEY,
  openaiApiKey: process.env.OPENAI_API_KEY,
});
```

<ParamField path="model" type="string" default="claude-sonnet-4-5-20250929">
  Main LLM model for taxonomy generation and reasoning.
</ParamField>

<ParamField path="fastLlm" type="string" default="claude-haiku-4-5-20251001">
  Faster model for document summarization to reduce costs.
</ParamField>

<ParamField path="sampleSize" type="number" default="100">
  Number of documents to sample for LLM labeling. Larger samples produce better taxonomies but cost more.
</ParamField>

<ParamField path="batchSize" type="number" default="200">
  Documents per minibatch during iterative clustering. Smaller batches = more refined taxonomies.
</ParamField>

<ParamField path="maxNumClusters" type="number" default="5">
  Maximum categories to generate. Start small (5-10) for focused taxonomies.
</ParamField>

<ParamField path="useCase" type="string">
  Custom description of your use case. Helps guide the model to generate relevant categories.
</ParamField>

## Usage Examples

### Basic Usage

```typescript
import { Delve } from '@delve/taxonomy';

const delve = new Delve();

// Pass strings directly
const result = await delve.run([
  'Fix the login button',
  'Add dark mode support',
  'How do I export my data?',
]);

console.log(`Generated ${result.taxonomy.length} categories`);
for (const category of result.taxonomy) {
  console.log(`- ${category.name}: ${category.description}`);
}
```

### With Document Objects

```typescript
import { Delve, Doc } from '@delve/taxonomy';

const documents: Doc[] = [
  { id: 'ticket-1', content: 'Cannot login after password reset' },
  { id: 'ticket-2', content: 'Request: Add CSV export feature' },
  { id: 'ticket-3', content: 'App performance is slow on mobile' },
];

const delve = new Delve({
  useCase: 'Categorize support tickets by type',
});

const result = await delve.run(documents);

// Each document now has a category
for (const doc of result.labeledDocuments) {
  console.log(`${doc.id}: ${doc.category}`);
}
```

### With Predefined Taxonomy

Skip taxonomy discovery and use your own categories:

```typescript
import { Delve } from '@delve/taxonomy';

const delve = new Delve();

const result = await delve.run(documents, {
  predefinedTaxonomy: [
    { id: '1', name: 'Bug', description: 'Bug reports and issues' },
    { id: '2', name: 'Feature', description: 'Feature requests' },
    { id: '3', name: 'Question', description: 'General questions' },
  ],
});
```

### One-off Generation

Use the convenience function for simple cases:

```typescript
import { generateTaxonomy } from '@delve/taxonomy';

const result = await generateTaxonomy(
  ['Fix bug', 'Add feature', 'Question about pricing'],
  {
    maxNumClusters: 3,
    useCase: 'Categorize feedback',
  }
);
```

## Result Structure

```typescript
interface DelveResult {
  /** Generated taxonomy categories */
  taxonomy: TaxonomyCategory[];

  /** All documents with assigned categories */
  labeledDocuments: Doc[];

  /** Run metadata and statistics */
  metadata: DelveMetadata;
}

interface TaxonomyCategory {
  id: string;
  name: string;
  description: string;
}

interface Doc {
  id: string;
  content: string;
  category?: string;
  summary?: string;
  explanation?: string;
}

interface DelveMetadata {
  numDocuments: number;
  numCategories: number;
  runDurationSeconds: number;
  categoryCounts: Record<string, number>;
  llmLabeledCount: number;
  warnings: string[];
}
```

## React Integration

```tsx
import { useState } from 'react';
import { Delve, DelveResult } from '@delve/taxonomy';

function TaxonomyGenerator() {
  const [result, setResult] = useState<DelveResult | null>(null);
  const [loading, setLoading] = useState(false);

  const generate = async (documents: string[]) => {
    setLoading(true);
    try {
      const delve = new Delve({ maxNumClusters: 5 });
      const result = await delve.run(documents);
      setResult(result);
    } finally {
      setLoading(false);
    }
  };

  return (
    <div>
      {loading && <p>Generating taxonomy...</p>}
      {result && (
        <div>
          <h3>Categories</h3>
          {result.taxonomy.map(cat => (
            <div key={cat.id}>
              <strong>{cat.name}</strong>: {cat.description}
            </div>
          ))}
        </div>
      )}
    </div>
  );
}
```

## Environment Variables

```bash
# Required
export ANTHROPIC_API_KEY="your-anthropic-key"

# Optional (for classifier embeddings)
export OPENAI_API_KEY="your-openai-key"
```

<Info>
  You can also pass API keys directly in the config:
  ```typescript
  new Delve({ anthropicApiKey: 'sk-...' })
  ```
</Info>

| LangGraph backend | Python | JavaScript |
